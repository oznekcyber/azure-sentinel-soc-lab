# Images

This directory contains images for the Azure Sentinel SOC Lab documentation.

## Contents

- `screenshots/` - Screenshots of the deployed SOC Lab
- `architecture-diagram.png` - Architecture diagram (to be added)

## Adding Screenshots

After deploying your lab and generating attack data:

1. Take screenshots of key Sentinel views
2. Save them in the `screenshots/` folder
3. Reference them in documentation using relative paths

Example:
```markdown
![Attack Map](images/screenshots/attack-map.png)
```
